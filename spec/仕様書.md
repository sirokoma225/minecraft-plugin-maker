# ChatGrass

## 概要
プレイヤーがチャットで「ｗ」を含むメッセージを送信した際に、その「ｗ」の個数に応じてプレイヤーの周囲に「草」を生成するプラグイン。草を植える場所が土の場合は草ブロックに置き換えます。ゲーム内でユニークな視覚効果やインタラクションを提供します。

## 主な機能
- プレイヤーがチャットで「ｗ」を含むメッセージを送信すると、その「ｗ」の個数に応じて周囲に「草」を生成。
- 草を植える場所が土の場合は草ブロックに置き換え。
- 生成される草の数や範囲は、「ｗ」の個数に比例して調整可能。

## 利用方法
- プレイヤーがチャットで「ｗ」を含むメッセージを送信する。

## 設定例（config.yml の主要キー）
```yaml
chatgrass:
  min_w_count: 1 # 最低限必要な'w'の数
  max_w_count: 10 # 最大反応する'w'の数
  generation_radius_per_w: 1 # 'w'1個あたりの生成半径増加量
  max_grass_blocks_per_w: 1 # 'w'1個あたりの最大草の数増加量
```

## MVP（最初に作る最小機能）
- プレイヤーがチャットで「ｗ」を含むメッセージを送信した際に、その「ｗ」の個数に応じてプレイヤーの足元に「草」を生成する。（例: 「ｗ」の数だけ草を生成）
- 生成場所が土の場合は草ブロックに置き換える。
